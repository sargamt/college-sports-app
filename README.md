# Playbook - College Sports Stats App

Installations:
```bash
cd college-sports-app
npm install
```

To start backend:
```bash
cd college-sports-app
cd flask-server
python server.py
```


To start frontend (separate terminal):
```bash
cd college-sports-app
npx expo start
```

For a virtual enviroment mac, run
cd flask-server
1) python3 -m venv venv
2) source venv/bin/activate
3) pip install flask
4) pip install requests
5) python3 server.py
